## 产品简介

AI Movie Maker 是一款基于浏览器的 AI 影片制作工具。你只需要输入一段剧本或故事文本，AI 将自动拆解角色、道具、场景与分镜，并通过 Seedance 视频生成模型将每个镜头转化为短视频片段，最终合成为一部完整的 AI 影片。

**核心能力：**
- AI 自动分析剧本，提取角色、道具、场景、分镜
- 可视化分镜编辑器，支持手动调整提示词和镜头参数
- 上传或 AI 生成角色/场景/道具参考图片
- 批量并发视频生成（默认并发 2 个任务，最多 10 个，更多可申请）
- 剪辑编辑器，支持转场效果
- 影片连续预览播放
- 项目自动保存至 Keepwork 云端工作区
- 支持多语言剧本

## 快速开始

1. 打开 AI Movie Maker 页面
2. **登录** Keepwork 账号
3. 点击 **"新建项目"** 按钮
4. 输入项目名称和剧本文本
5. 选择美术风格、画面比例等设置
6. 点击 **"AI 分析剧本"**，AI 将自动拆解出角色、道具、场景和镜头
7. 检查和编辑分镜内容 — 调整提示词、上传或生成参考图片
8. 点击 **"开始生成视频"** 批量生成所有短片
9. 生成完成后，进入 **"预览影片"** 页面观看你的 AI 电影

## 剧本与设置

### 输入剧本

进入项目后，在左侧树形导航中点击 **"📄 剧本 & 设置"** 节点。右侧面板显示：

- **剧本 / 故事大纲** 文本区域：输入或粘贴你的剧本文本
- 系统支持任意长度的故事文本
- 可以随时修改剧本并重新运行 AI 分析

### 项目设置

在剧本面板下方展示可调参数：

| 参数 | 说明 |
|------|------|
| 时长（分钟） | 目标总视频时长（1-90 分钟） |
| 集数 | 总集数（1-99） |
| 比例 | 画面宽高比：16:9、9:16、1:1、4:3、3:4 |
| 片段时长 | 每个短片的默认时长：5-10 秒 |
| 模型 | 视频生成模型：Seedance 2.0 或 Seedance 2.0 Fast |
| 美术风格 | 预设风格或自定义 |
| 时代环境 | 环境预设或自定义 |
| 人物族裔 | 族裔预设或自定义 |
| 生成音频 | 是否为每个片段生成旁白音频 |
| 旁白音色 | 选择 TTS 配音音色 |
| 旁白语速 | 调整旁白朗读速度 |

### 美术风格预设

| 预设名称 | 说明 |
|----------|------|

### 时代环境预设

| 预设 | 说明 |
|------|------|

支持东亚人、东南亚人、南亚人、黑人、白人、拉丁裔、中东人、混血/多元和自定义选项。

支持 7 种语言：中文、English、日本語、한국어、Français、Español、Deutsch。选择后，AI 生成的所有文本内容（标题、概要、角色名、场景名、旁白等）都会使用对应语言。

### 提示词预设版本

提示词预设控制发送给 LLM 的系统提示词模板版本，可在新建项目或项目设置中选择。不同预设针对不同语言和生成质量进行了优化。

## AI 剧本分析

在剧本输入完成后，点击 **"AI 分析剧本"** 按钮：

1. AI 根据剧本内容和目标时长，自动拆解出：
   - **角色列表**：角色名称和视觉外貌描述
   - **道具列表**：重要物品、武器、载具等
   - **场景列表**：拍摄场地的描述
   - **分镜列表**：按时间顺序的短片镜头，每个 5-15 秒
2. AI 会根据总时长计算需要多少个短片来填满
3. 分析完成后自动进入分镜编辑视图
4. 可以点击 **"查看 AI 请求"** 按钮预览将要发送给 LLM 的提示词
5. 支持流式输出，实时显示分析进度

## 分镜编辑器

分镜编辑器是 AI Movie Maker 的核心工作区，采用左右双栏布局：

### 左侧树形导航

左侧边栏显示项目的完整结构树：

- **📄 剧本 & 设置** — 剧本文本和项目参数
- **📝 概要** — 故事概要
- **👥 角色** — 所有角色列表，支持文件夹分组
- **🎒 道具** — 所有道具列表，支持文件夹分组
- **🎬 场景** — 所有场景列表，支持文件夹分组
- **📋 分镜** — 所有短片镜头列表，支持文件夹分组

点击树节点在右侧显示详细编辑面板。侧边栏宽度可拖拽调整。

### 角色管理

点击 **"👥 角色"** 分组节点，右侧显示角色画廊视图：

- 以卡片形式展示所有角色
- 每张卡片显示角色名称、描述和参考图片
- 支持的操作：
  - **添加角色**：手动创建新角色
  - **新建文件夹**：创建文件夹分组角色
  - **重新生成**：AI 重新生成角色描述
  - **生成缺失图片**：批量为缺少图片的角色生成 AI 参考图
- 点击单个角色卡片进入详细编辑：
  - 编辑角色名称和描述
  - 上传/生成参考图片
  - 查看和管理候选图片
  - 设置锚定图片

### 道具管理

与角色管理类似，支持所有道具的画廊展示和编辑：

- 以卡片形式展示道具
- 支持添加、删除、重新生成、生成缺失图片
- 支持文件夹分组
- 每个道具可设置名称、描述和参考图片

### 场景管理

场景画廊使用宽卡片布局，展示更多场景信息：

- 显示场景名称、描述、参考图片
- 显示场景元数据标签：光照、时间段、天气、氛围
- 支持添加、删除、重新生成、生成缺失图片
- 支持文件夹分组

### 分镜（短片）管理

分镜画廊以宽卡片展示每个镜头：

- 显示镜头编号、视频/图片缩略图
- 显示状态标签：待处理 / 生成中 / 已完成 / 失败
- 显示所在场景、出场角色
- 支持添加、删除、重新生成
- 支持文件夹分组

点击单个分镜卡片进入详细编辑：

- 编辑视频生成提示词（prompt）
- 设置镜头类型、情绪、摄影机运动、摄影机角度
- 关联角色和道具
- 设置所在场景
- 调整片段时长
- 上传参考图片或参考视频
- 选择模型覆盖（可单独为某个片段指定不同模型）
- 开关水印
- 查看生成任务详情

### 文件夹分组

角色、道具、场景和分镜都支持文件夹分组：

- 在分组标题栏点击 **📁** 按钮创建新文件夹
- 通过拖放将项目移入文件夹
- 文件夹可重命名或删除
- 未分组的项目显示在"未分组"区域

### 拖放排序

- 分镜和其他项目支持拖拽排序
- 可将项目拖入文件夹或从文件夹中拖出
- 拖放时有视觉反馈

## 参考图片

### 上传参考图片

- 在角色、道具或场景详情中点击上传区域
- 支持 JPG、PNG、WebP 格式
- 图片上传至七牛临时存储
- 上传后 URL 保存在项目文件中

### AI 生成参考图片

- 点击 **"🎨 AI 生成图片"** 按钮
- 根据角色/道具/场景的文字描述生成参考图
- 使用即梦（Jimeng）图片生成模型
- 生成结果可作为参考图使用

### 锚定图片验证

角色和道具支持锚定图片（Anchor Image）：

- 锚定图片是该角色/道具的标准参考外观
- 锚定验证通过后显示 ✅ 标记
- 有助于在视频生成中保持角色一致性

### 候选图片

- 每次 AI 生成或手动上传的图片会加入候选列表
- 可从候选列表中选择最满意的图片作为主参考图
- 候选图片持久保存在项目中

## 镜头参数设置

每个分镜（短片）可以设置详细的电影摄影参数，这些参数会被纳入视频生成提示词中：

### 镜头类型

| 选项 | 说明 |
|------|------|

| 独白 (monologue) | 独白场景 |
| 反应 (reaction) | 反应镜头 |
| 互动 (interaction) | 角色互动 |
| 魔法/特效 (magic) | 魔法与特效 |
| 战斗 (battle) | 战斗场面 |
| 追逐 (chase) | 追逐戏 |
| 过渡 (transition) | 过渡镜头 |
| 蒙太奇 (montage) | 蒙太奇 |
| 插入特写 (insert) | 插入特写 |
| 闪回 (flashback) | 闪回片段 |

### 情绪氛围

提供 15 种情绪预设：恐惧/不安、威严/力量、神秘/好奇、孤独/渺小、温暖/友情、愤怒/冲突、觉醒/顿悟、仪式/庄严、追逐/逃亡、悲伤/失落、欢乐/庆祝、紧张/悬疑、宁静/平和、惊讶/震撼、浪漫/柔情。

### 摄影机运动

提供 19 种运动预设：

- 固定机位 (Static)
- 慢推/快推 (Dolly In)
- 慢拉/快拉 (Dolly Out)
- 希区柯克变焦 (Dolly Zoom In)
- 慢升/慢降 (Crane Up/Down)
- 平滑左摇/右摇 (Pan)
- 慢仰/慢俯 (Tilt)
- 弧形左移/右移 (Arc)
- 手持晃动 (Handheld)
- 稳定跟踪 (Tracking)
- 慢变焦推近/拉远 (Zoom)
- 慢环绕 (Orbit)

### 摄影机角度

提供 14 种角度预设：

- 平视 (Eye-level)
- 仰拍/俯拍 (Low/High Angle)
- 荷兰角 (Dutch Angle)
- 鸟瞰/蚁视 (Bird's Eye / Worm's Eye)
- 过肩 (Over-the-shoulder)
- 第一人称视角 (POV)
- 特写/大特写 (Close-up / Extreme Close-up)
- 中景/全景/远景 (Medium / Wide / Extreme Wide)
- 双人镜头 (Two-shot)

### 光照预设

每个分镜可单独设置光照条件，与场景的时间段、天气、氛围属性配合使用。

## AI 增强功能

### 增强角色描述

- 点击角色组的 **"🔄 重新生成"** 按钮
- AI 根据剧本上下文和已有信息，为所有角色生成更详尽的视觉外貌描述
- 支持流式输出，实时查看增强过程

### 增强场景描述

- 点击场景组的 **"🔄 重新生成"** 按钮
- AI 为所有场景生成更详细的环境设定描述
- 自动补充光照、时间、天气、氛围等元数据

### 增强分镜提示词

- 点击分镜组的 **"🔄 重新生成"** 按钮
- AI 根据角色描述、场景设定和美术风格，为每个镜头生成详细的视频生成提示词
- 增强后的提示词包含完整的角色外观、场景布局和镜头说明

### 预检查

- 在生成视频前，可以运行 **预检查**（Preflight）
- AI 检查所有分镜是否存在问题：
  - 提示词是否过短或缺失
  - 角色和场景引用是否有效
  - 参考图片是否缺失
  - 给出修复建议

## 视频生成

### 开始生成

1. 确保已登录且所有分镜已编辑完毕
2. 点击 **"开始生成视频"** 按钮
3. 系统将所有待处理和失败的分镜加入生成队列
4. 每个分镜的当前视频（如果有）会被保存为候选视频
5. 默认同时运行 **2 个**生成任务，最多 **10 个**，更多可申请

### 生成进度与状态

生成视图以网格卡片展示所有分镜的状态：

| 状态 | 图标 | 说明 |
|------|------|------|
| 待处理 | ⬜ | 在队列中等待 |
| 生成中 | ⏳ | 正在调用 Seedance API |
| 已完成 | 🎬 | 生成成功，可预览 |
| 失败 | ❌ | 生成出错，显示错误信息 |

- 页面顶部显示整体进度条
- 实时刷新各片段状态
- 后台通过轮询检测任务完成情况

### 重试失败片段

- 点击 **"重试失败"** 按钮，将所有失败片段重置为待处理并重新加入队列
- 也可以点击单个片段的重试按钮单独重试

### 停止生成

- 点击 **"⏹ 停止生成"** 按钮可中断所有正在进行的任务
- 已提交但未完成的任务将标记为失败并显示"已手动停止"

### 视频生成统计

- 在生成页面顶部显示视频生成统计：
  - 总提交次数
  - 成功次数
  - 失败次数
  - 累计视频时长
- 点击 **"详情"** 查看每次生成任务的详细记录

## 剪辑编辑器

### 时间线播放

- 点击 **"🎬 剪辑编辑器"** 进入
- 基于 Canvas 的时间线播放器
- 按顺序播放所有短片
- 没有视频的片段显示黑色占位画面（附文字描述）
- 支持时间线跳转到指定片段

### 转场效果

剪辑编辑器支持 4 种片段间转场效果：

| 转场 | 说明 |
|------|------|
| 直切 (Cut) | 无过渡，直接切换（默认） |
| 淡入淡出 (Fade Black) | 先淡出到黑色再淡入 |
| 交叉溶解 (Crossfade) | 前后片段交叉叠化 |
| 左擦除 (Wipe Left) | 从右到左擦除过渡 |

转场动画持续 600 毫秒。

## 旁白与配音

### 音色选择器

- 在项目设置中点击旁白音色选择器
- 弹出 **音色浏览器** 模态框
- 支持按分类浏览和搜索豆包 TTS 音色库
- 支持实时试听（通过 WebSocket 连接 Volcano TTS 服务）
- 选择后音色类型 ID 保存到项目设置中

### 旁白语速与语言

- **语速调整**：可通过滑块调整旁白朗读速度
- **语言设置**：控制旁白生成使用的语言
- 当启用音频生成时，每个短片的旁白文本会被自动转为语音

## MP4 转 WebP / 帧序列

AI Movie Maker 内置了 MP4 视频转换工具：

- **动画 WebP 导出**：将短片 MP4 转换为动画 WebP 格式
  - 支持背景移除（通过 MediaPipe/ONNX 人像分割模型）
  - 可调整画质、帧率
  - 使用 WASM WebP 编码器（libwebp）
- **帧序列导出**：将视频拆分为 PNG/JPG 帧序列
  - 可自定义输出帧率

## 工作区与云端同步

- 所有项目数据保存为 `项目名.aimovie.md` 文件
- 存储在 Keepwork PersonalPageStore 的 `AIMovieMaker` 工作区中
- 数据通过三层缓存：内存 → localStorage → IndexedDB
- 每 5 秒自动批量同步至云端（Git 存储）
- 可在菜单中查看工作区文件浏览器
- 生成任务日志保存在 `logs/` 子目录中

## 常见问题 FAQ

### Q: 为什么需要登录？
A: AI Movie Maker 使用 Keepwork 账号体系进行身份验证，项目文件保存在你的个人 Keepwork 工作区中，需要登录才能访问。

### Q: 视频生成需要多长时间？
A: 每个 5-10 秒的短片通常需要 1-3 分钟生成。系统默认同时生成 2 个片段，最多可调至 10 个（更多可申请），总时间取决于片段数量和并发数。

### Q: 上传的图片保存在哪里？
A: 参考图片上传到七牛临时存储服务，URL 保存在项目文件中。临时存储有过期时间，建议及时完成生成。

### Q: 可以重新生成单个片段吗？
A: 可以。在分镜详情中点击 🔄 按钮可以重新生成单个片段的提示词。在生成页面可以对失败的片段单独重试视频生成。

### Q: 支持最多多少个分镜？
A: 单个项目最多支持 **50 个分镜**。

### Q: 项目数据会丢失吗？
A: 项目数据通过 Keepwork 云端同步保存，同时在浏览器本地有三层缓存（内存、localStorage、IndexedDB）。正常使用不会丢失数据。

### Q: 如何切换 AI 模型？
A: 在全局设置中可以切换 LLM 模型（用于剧本分析、增强等文本任务）。视频生成模型（Seedance）可在项目设置中选择。

### Q: 生成的视频有水印吗？
A: 默认无水印。可在单个分镜设置中手动开启水印选项。

### Q: 支持哪些浏览器？
A: 推荐使用 Chrome、Edge 等现代浏览器。需要支持 ES Modules、Canvas API 和 HTML5 Video。部分高级功能（WebP 编码、ONNX 推理）需要 WASM 和 WebGPU 支持。
